# spring-jms

This project provides following simple examples based on spring jms framework -  
- Simple spring JMS based message browser to a Queue: net.neophyte.messaging.spring.jms.p2p.BrowserApp.java
- Simple spring JMS based message producer to a Queue: net.neophyte.messaging.spring.jms.p2p.ProducerApp.java 
- Simple spring JMS based message consumer on a Queue: net.neophyte.messaging.spring.jms.p2p.ConsumerApp.java
- Simple spring JMS based message publisher to a Topic: net.neophyte.messaging.spring.jms.pubsub.PublisherApp.java
- Simple spring JMS based non-durable Topic subscriber and message consumer on a Topic: net.neophyte.messaging.spring.jms.pubsub.SubscriberApp.java
- Simple spring JMS based durable Topic subscriber and message consumer on a Topic: net.neophyte.messaging.spring.jms.pubsub.DurableSubscriberApp.java
